# 🛒 Sistema de Vendas

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido em Python com o objetivo de simular um sistema de vendas, permitindo o cadastro e gerenciamento de produtos, além do registro de operações de venda através de uma interface em terminal.

O projeto foi criado para praticar conceitos fundamentais de programação e lógica, aplicando-os em um cenário próximo ao utilizado em sistemas comerciais reais.

## 🎯 Funcionalidades

* Cadastro de produtos.
* Consulta de produtos cadastrados.
* Listagem de produtos.
* Registro de vendas.
* Controle de estoque.
* Cálculo de valores de venda.
* Armazenamento de informações em arquivos.

## 📚 Conceitos Praticados

* Variáveis e tipos de dados.
* Estruturas condicionais (`if`, `elif` e `else`).
* Estruturas de repetição (`while` e `for`).
* Manipulação de arquivos.
* Funções.
* Entrada e saída de dados.
* Listas e coleções.
* Lógica de programação.
* Organização de sistemas em menus.
